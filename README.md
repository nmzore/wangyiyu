**git命令**

```

git add . # 加入本地缓存区
git commit -m '提交' # 提交
git rebase  # 变基
git reset logid # 回退
git merge xx # 合并
git push # 推送
git pull # 拉取
git checkout -b # 创建一个新的分支比切换过去
git branch # 查看所有的峰值
```
